Create premium, modern, non-generic mobile app interfaces and flows.

Use this skill when the user asks for:
- mobile app screens
- onboarding
- profile screens
- dashboards for mobile
- messaging UI
- activity feeds
- finance apps
- health apps
- productivity apps
- e-commerce app flows
- React Native or Expo mobile UI

Design mobile interfaces with native-feeling priorities:
- clear hierarchy within smaller space
- touch-friendly interactions
- focused screen purpose
- reduced clutter
- intentional component stacking
- strong rhythm between dense and calm areas

Avoid:
- web-style layouts compressed into phone dimensions
- generic card stacks with weak hierarchy
- repetitive tab/dashboard structures
- overly decorative interfaces that ignore usability
- screens that feel interchangeable across products

When generating concepts, vary:
- navigation patterns
- content grouping
- information density
- interaction emphasis
- emotional tone
- gesture opportunities
- screen pacing

Every screen should have a clear primary action and a clear reason for its layout.
