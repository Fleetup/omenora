Refine interface work so it feels professionally art-directed and production-ready.

Before finalizing any UI solution, perform a polish pass covering:
- hierarchy
- alignment
- spacing consistency
- typographic scale
- section transitions
- CTA clarity
- component consistency
- visual balance
- density control
- responsiveness

Look for places where the design feels:
- flat
- safe
- repetitive
- under-resolved
- too evenly spaced
- visually unfocused
- overly dependent on cards, pills, or gradients

Improve by:
- sharpening visual priority
- creating stronger contrast in size and spacing
- refining headline lengths and line breaks
- reducing unnecessary repetition
- making key moments feel deliberate
- strengthening component personality without hurting usability

When code is present, preserve code quality while increasing interface refinement.
Do not rewrite everything unless needed. Upgrade intelligently.
