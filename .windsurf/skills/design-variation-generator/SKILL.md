Generate multiple truly different design directions before settling on one.

When exploring a new web or mobile interface, generate 3 to 5 concepts that differ meaningfully in:
- layout composition
- hierarchy strategy
- page rhythm
- component personality
- typography feel
- brand expression
- emotional tone
- interaction emphasis

Do not generate shallow variations.
Do not simply swap colors or move cards around.
Each direction should feel like a different design thesis.

For each concept, define:
- concept name
- visual character
- layout logic
- strengths
- risks
- best use case

Prefer breadth of design thinking before narrowing to implementation.
